# Mission: Build a Flappy Bird game in HTML

## Why
Build a real, playable Flappy Bird clone that runs in a web browser -- a fun side project to learn the fundamentals of HTML, CSS, and JavaScript from scratch.

## Success looks like
- A working game with a bird that flaps, scrolling pipes, and a score counter
- The game runs by opening a single HTML file in any browser
- The user understands each line of code they wrote (not just copy-paste)

## Constraints
- No prior coding experience -- every concept must be introduced from scratch
- No frameworks or build tools -- pure HTML, CSS, and vanilla JavaScript
- Single HTML file (inline CSS and JS) for simplicity

## Out of scope
- Mobile responsiveness or touch controls (desktop focus)
- Leaderboards, backends, or persistence
- Advanced game engine patterns
