# Flappy Bird (HTML Game) Resources

## Knowledge

- [MDN: Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)
  The authoritative reference for the HTML5 `<canvas>` element. Use for: all drawing operations, animation frames, rendering.

- [MDN: JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)
  Mozilla's comprehensive JS tutorial. Use for: language fundamentals as they come up (variables, functions, events, objects).

- [Eloquent JavaScript (online book)](https://eloquentjavascript.net/)
  Free, interactive book. Chapters 2--4 and 16 cover the exact concepts needed here. Use for: deeper explanation of JS concepts introduced in lessons.

- [MDN: requestAnimationFrame](https://developer.mozilla.org/en-US/docs/Web/API/window/requestAnimationFrame)
  Reference for the browser's animation loop. Use for: understanding game loop timing.

## Wisdom (Communities)

- [r/learnjavascript](https://reddit.com/r/learnjavascript)
  Beginner-friendly subreddit for asking JS questions.

- [r/gamedev](https://reddit.com/r/gamedev)
  General game development community. Broad but useful for design patterns and inspiration.

## Gaps
- No single high-quality resource covers "vanilla JS game from scratch for complete beginners." The lessons fill this gap.
